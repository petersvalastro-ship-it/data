# Brain Blueprint: Architects of the Mind — Workbook Suite

A scaffolded **grades 3–8** curriculum package (three bands: **3–4**, **5–6**, **7–8**) aligned with Peter Valastro’s **Library & media** resources and the Canva design [*Brain Blueprint: Architects of the Mind*](https://www.canva.com/design/DAHFA_PfHjs/unckjhxwPRJzvvhIZWCXjA/view?utm_content=DAHFA_PfHjs&utm_campaign=designshare&utm_medium=link&utm_source=editor).

## On your website

After you run **`npm run sync-workbook`** from the `cursor_project` root, the same files are served at **`/workbook/`**. Open the **Brain Blueprint workbooks** page from the site menu to browse every document in the browser.

## How to open this project

1. In **Cursor** (or VS Code): **File → Open Folder…**
2. Choose: `architects-of-the-mind-workbook`  
   Full path on your machine:  
   `cursor_project/architects-of-the-mind-workbook`  
   (inside your existing `cursor_project` folder on the Desktop)

3. Start reading from **`README.md`** (this file), then open:
   - `cross-grade/unit-at-a-glance.md`
   - `teacher-guide/overview.md`
   - `grades-3-4/student-workbook.md` (and 5–6, 7–8)

Markdown previews: open any `.md` file and use the editor’s Markdown preview (e.g. **Open Preview** in Cursor).

## What’s inside

| Folder / file | Purpose |
