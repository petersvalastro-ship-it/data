# How we work together (quick reference)

This file is for **you**. The AI also follows `.cursor/rules/user-communication-preferences.mdc` automatically.

---

## What you can expect from the assistant

- It will **double-check** it understood your request and **stick to it**.
- When a task is **finished**, you always get a **short end summary** written for someone new to this.
- Summaries split **what the assistant did** vs **what you still need to do** (if anything).

---

## How to spot “my job” vs “your job” in replies

Look for these headings (or similar):

- **Done for you** / **Assistant** → the AI handled it.
- **Your next steps** / **You** → you still need to do something.

If there is nothing for you, it should say that clearly.

---

## Bigger, easier reading

The chat font size is controlled by **Cursor settings**, not by the assistant. For easier reading: increase editor / UI zoom in Cursor, or use headings and bullets in the summary (the rule asks the assistant to do that).

---

## Want this on every project?

Copy `.cursor/rules/user-communication-preferences.mdc` into another repo’s `.cursor/rules/` folder, or merge its ideas into your global Cursor user rules.
