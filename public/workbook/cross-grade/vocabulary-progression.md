# Vocabulary progression — 3–4 → 5–6 → 7–8

Use these as **word walls** and **assessment targets**. Define in student-friendly language first; add precision in older bands.

## Grades 3–4 (everyday language + seeds)

- Brain, think, remember, focus, feeling, calm, habit  
- Message (signal), sense (see/hear/feel), rest, practice  
- **Tool** vs **trap** (for tech experiences—operational, not moral judgment on people)

## Grades 5–6 (introduce formal terms with metaphors)

- **Neuron**, **synapse** (locks and keys / bridges)  
- **Prefrontal cortex** — “decision headquarters”  
- **Amygdala** — “alarm system” (simplified; not destiny)  
- **Neuroplasticity** — brain changes with practice  
- **Stimulus**, **response**, **habit loop** (cue → routine → reward) at a basic level  
- **Digital wellness**, **notification**, **algorithm** (introductory)

## Grades 7–8 (precision + ethics)

- **Executive function:** working memory, inhibitory control, cognitive flexibility  
- **Dopamine** — prediction, learning, motivation; **not** “only pleasure”  
- **Variable reward**, **attention economy**, **persuasive design** (ethical critique)  
- **Source evaluation**, **bias**, **privacy**, **data**  
- **Sleep hygiene**, **circadian** (general; no diagnosing)

## Words to avoid or qualify

- Avoid implying students are “addicted” unless following district mental-health protocols.  
- Avoid brain **myths** (e.g. “you only use 10%”).  
- Use **“can influence”** rather than **“causes”** when linking tech to brain/behavior unless citing age-appropriate sources in 7–8.
