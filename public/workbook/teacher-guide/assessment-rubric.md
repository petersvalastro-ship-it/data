# Culminating rubric — adaptable (all bands)

Use **one row set**; adjust evidence type by band (poster vs slides vs paper).

| Criteria | Emerging (1) | Proficient (2) | Strong (3) |
|----------|--------------|----------------|------------|
| **Brain understanding** | Vague or inaccurate | Names a real function or part in student language | Accurate + connects to **experience** or **behavior** |
| **SEL connection** | Missing | Names one strategy | Strategy + **when** to use it |
| **Technology insight** | Only likes/dislikes | Tool vs trap with one example | Example + **plan** or **design awareness** |
| **Communication** | Hard to follow | Clear for peers | Clear + organized + cites source (5–8 as assigned) |

**Integrity:** Audits and reflections should not be graded for “good answers”—grade **effort**, **specificity**, and **revision**.
